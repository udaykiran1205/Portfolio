## 🚀 Installation & Setup
1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/Portfolio.git
   ```
2. Navigate to the project directory:
   ```sh
   cd portfolio-website
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm start
   ```
